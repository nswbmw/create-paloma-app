## create-paloma-app

> Create [Paloma](https://github.com/palomajs/paloma) app easily.

### Directory

```
.
├── app/
│   ├── controller/
│   │   └── user.js
│   ├── lib/
│   │   ├── mongo.js
│   │   └── redis.js
│   ├── middleware/
│   ├── public/
│   ├── route/
│   │   └── users.js
│   ├── service/
│   │   └── User.js
│   └── view/
├── config/
│   ├── default.js
│   └── production.js
├── script/
├── __tests__/
├── .editorconfig
├── .eslintignore
├── .eslintrc.json
├── .gitignore
├── package.json
├── pm2.config.js
├── app.js
└── README.md
```

### Usage

```sh
$ git clone https://github.com/palomajs/create-paloma-app.git
$ cd create-paloma-app && npm i
$ npm start
```
